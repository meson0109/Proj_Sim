# Proj_Sim
=======
# CNN experiment files

This folder contains the CNN-only part of the visual recognition simulation.

- `cnn_course_blocks.py`: copied/adapted Project3-style CNN layers, optimizer, and solver.
- `mnist_cnn.py`: PyTorch CNN pipeline for MNIST recognition with 5:1:1 splitting.
- `mnist_draw_ui.py`: interactive 28x28 drawing pad with real-time 0-9 probability bars.
- `detection_cnn.py`: Pascal VOC 2007 to YOLO conversion, YOLO training, mAP validation, and test-folder prediction.
- `cnn_main_experiment.ipynb`: main notebook for the report outputs.

The SNN part is intentionally left out for now.
